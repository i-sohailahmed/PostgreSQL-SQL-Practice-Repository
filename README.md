# PostgreSQL SQL Practice Repository

## Project Overview

This project documents my SQL learning journey using PostgreSQL.

The objective was to build a strong foundation in SQL by practicing every major concept, from basic CRUD operations to advanced Window Functions. Each topic was practiced using self-created sample tables and real SQL queries to understand how SQL is used in real-world databases.

This repository serves as my personal SQL reference as well as proof of hands-on SQL practice.
